## PICO 4 简介

### 技术规格
| 参数 | 规格 |
|------|------|
| 分辨率 | 单眼2160×2160 (4K+) |
| 刷新率 | 72Hz / 90Hz |
| 视场角 (FOV) | 105° |
| 处理器 | Snapdragon XR2 |
| 内存 | 8GB RAM + 128/256GB 存储 |
| 重量 | 约304g |

### 包装内容
- PICO 4 VR头显
- 左右手控制器 x2
- 数据线 (USB-C)
- 充电适配器
- 鼻托和眼镜支架

## 首次开机设置

### 步骤1：激活设备
1. 按住电源键3秒开机
2. 选择语言（支持中文）
3. 连接WiFi网络

### 步骤2：账号绑定
1. 使用手机扫描屏幕二维码
2. 下载 **PICO VR** App
3. 注册/登录PICO账号
4. 完成设备绑定

### 步骤3：安全区域设置
1. 选择合适的边界系统：
   - **Stationary（固定）**：适合坐着使用
   - **Room-Scale（房间规模）**：适合站立移动
2. 跟随指引绘制安全边界

### 步骤4：控制器校准
1. 按照屏幕提示握持控制器
2. 完成追踪空间校准
3. 测试按键功能

## 开发者模式配置

### 启用USB调试

#### 方法1：通过PICO设置
1. 进入 **设置 → 关于本机**
2. 连续点击 **版本号** 7次
3. 返回 **设置 → 开发者选项**
4. 开启 **USB调试**

#### 方法2：通过ADB命令
```bash
# 连接设备后执行
adb shell settings put global development_settings_enabled 1
adb shell settings put global adb_enabled 1
```

### 验证连接
```bash
adb devices
```
应该显示类似：
```
List of devices attached
XXXXXXXX    device
```

## USB连接与文件传输

### 连接步骤
1. 使用USB-C数据线连接PICO到电脑
2. PICO会弹出"允许USB调试"提示，勾选"始终允许"
3. 点击"确定"

### 文件传输位置
PICO 4 的视频存储路径：
```
/sdcard/Movies/
/sdcard/DCIM/Camera/
/sdcard/PICOVideos/
```

### 使用本项目自动同步
```bash
python main.py sync
```
或运行监控服务：
```bash
python sync_to_pico.py
```

## 观看VR视频

### 支持的视频格式
| 格式 | 支持 | 备注 |
|------|------|------|
| MP4 (Equirectangular) | ✅ 推荐 | 标准全景格式 |
| MKV | ✅ | 需要内置解码器 |
| AVI | ⚠️ | 部分支持 |

### 观看方法

#### 方法1：使用PICO视频App
1. 将视频复制到 `/sdcard/Movies/`
2. 打开 **PICO视频** 应用
3. 在"本地"标签页找到视频
4. 点击播放

#### 方法2：使用第三方播放器
推荐应用：
- **Skybox VR Player** - 功能强大
- **DeoVR** - 高质量播放
- **Pigasus** - 轻量级

#### 方法3：直接打开
1. 打开 **文件管理器**
2. 导航到视频所在目录
3. 点击视频文件即可播放

### 视频播放设置
- **投影模式**：选择 "360° Equirectangular"
- **分辨率**：通常自动适应
- **3D模式**：如需3D效果选择相应选项

## 故障排除

### Q1: 设备无法被电脑识别？
**解决方案**:
1. 检查USB线是否为数据线（非仅充电）
2. 尝试更换USB端口
3. 确认已开启USB调试
4. 重启设备和电脑

### Q2: 视频无法播放？
**解决方案**:
1. 检查视频编码是否为H.264/H.265
2. 确认是等距柱状投影格式
3. 尝试降低视频分辨率
4. 使用其他播放器尝试

### Q3: 画面模糊？
**可能原因及解决**:
1. **镜头脏了** - 清洁镜片
2. **IPD未调整** - 调整瞳距旋钮
3. **视频分辨率低** - 使用更高分辨率源文件
4. **刷新率问题** - 设置中调整为90Hz

### Q4: 追踪丢失？
**解决方案**:
1. 确保环境光线充足
2. 移除反光物体
3. 清洁头显外部的传感器
4. 重新进行追踪校准
