## AI内容生成器简介

本项目的AI内容生成模块可以帮你快速创建VR相关的内容素材，包括：
- 🎨 **VR场景模板** - 用于构建虚拟世界
- 🎭 **虚拟形象(Avatar)** - VRChat或元宇宙中的角色
- 📖 **VR故事剧本** - 互动式叙事内容
- 💻 **Unity代码** - 快速启动开发项目

### 如何使用

#### 方法1：通过主程序
```bash
python main.py ai
```

#### 方法2：单独运行各模块
```bash
# 生成场景
python ai_content/vr_scene_generator.py

# 生成Avatar
python ai_content/avatar_generator.py

# 生成故事
python ai_content/story_generator.py
```

## VR场景生成

### 支持的主题
| 主题 | 描述 | 适用场景 |
|------|------|----------|
| forest | 神秘森林 | 冒险、探索类 |
| space | 太空站 | 科幻、未来感 |
| beach | 热带海滩 | 放松、休闲类 |

### 生成的输出
每个场景会生成两个文件：

#### 1. JSON描述文件 (`ai_content/generated/`)
```json
{
  "name": "神秘森林",
  "environment": {
    "lighting": "柔和的阳光透过树冠",
    "atmosphere": "宁静、神秘"
  },
  "objects": [
    {"type": "tree", "count": 50},
    {"type": "rock", "count": 20}
  ],
  "interactive_elements": [
    {"type": "teleport_pad", "positions": 5}
  ]
}
```

#### 2. Unity脚本 (`ai_content/generated/`)
```csharp
public class AutoGeneratedScene : MonoBehaviour
{
    void Start()
    {
        Debug.Log("加载场景: 神秘森林");
        SetupLighting();
        SpawnObjects();
    }
}
```

### 自定义主题
编辑 `ai_content/vr_scene_generator.py` 中的 `templates` 字典来添加新主题。

## 虚拟形象生成

### 生成内容
每次运行会随机生成一个完整的Avatar配置，包括：

#### 外观特征
- 发型（颜色、样式）
- 眼睛颜色
- 肤色
- 面部特征（眼镜、胡须、妆容）

#### 服装搭配
- 上衣、下装、鞋子
- 配饰（手表、项链、帽子等）

#### 性格特质
随机分配3个性格特点（友好、冒险、创意等）

### 输出文件
保存至 `ai_content/avatars/` 目录：
```
avatar_20260406_120000.json
```

### 示例输出
```json
{
  "style": "realistic",
  "gender": "female",
  "appearance": {
    "hair": {
      "color": "pink",
      "style": "long"
    },
    "eyes": { "color": "blue" }
  },
  "personality_traits": ["friendly", "creative", "playful"]
}
```

### 批量生成
```python
from ai_content.avatar_generator import AvatarGenerator

generator = AvatarGenerator()
avatars = generator.batch_generate(10)  # 生成10个Avatar
```

## VR故事生成

### 支持的流派
| 流派 | 场景设定 | 目标 |
|------|----------|------|
| adventure | 古代遗迹 | 寻找宝藏 |
| mystery | 维多利亚豪宅 | 解开谋杀之谜 |
| fantasy | 魔法森林 | 拯救王国 |
| sci-fi | 未来太空站 | 阻止外星入侵 |

### 故事结构
生成的故事遵循三幕式结构：

#### 第一幕：引入
- 到达目的地
- 与NPC对话
- 了解任务目标

#### 第二幕：发展
- 面对挑战/障碍
- 解决谜题
- 收集线索/物品

#### 第三幕：高潮
- 最终对决
- 达成目标
- 获得奖励

### VR交互元素
每个故事都包含适合VR的交互设计：
- 物品拾取
- 谜题解密
- NPC对话
- 环境探索
- 传送移动

### 输出文件
- JSON数据: `ai_content/stories/`
- Unity控制器脚本: `ai_content/stories/`

## Unity脚本生成

### 自动生成的脚本类型

#### 1. 场景脚本
```csharp
// AutoGenerated{SceneName}.cs
public class AutoGeneratedScene : MonoBehaviour
{
    void Start() { /* 初始化 */ }
}
```

#### 2. 故事控制器
```csharp
// VRStory_{StoryID}.cs
public class VRStoryController : MonoBehaviour
{
    public void CompleteCurrentAct() { /* 推进剧情 */ }
}
```

### 如何使用生成的脚本
1. 将 `.cs` 文件复制到Unity项目
2. 创建对应的GameObject
3. 挂载脚本组件
4. 根据JSON描述填充具体内容

## 自定义扩展

### 添加新场景主题
```python
# 在 vr_scene_generator.py 中添加
templates["cyberpunk"] = {
    "name": "赛博朋克城市",
    "description": "霓虹灯闪烁的未来都市",
    "environment": {
        "lighting": "霓虹灯和全息投影",
        "atmosphere": "高科技、阴暗"
    },
    "objects": [
        {"type": "building", "count": 30, "style": "futuristic"},
        {"type": "neon_sign", "count": 50, "colors": "pink, blue, purple"}
    ]
}
```

### 添加新故事流派
```python
# 在 story_generator.py 中添加
settings["horror"] = {
    "setting": "废弃的精神病院",
    "goal": "逃离恐怖的地方",
    "obstacles": ["幽灵", "谜题", "黑暗"],
    "tone": "恐怖紧张"
}
```

可以使用以下API增强生成能力：
- OpenAI GPT - 自然语言描述
- Midjourney/DALL-E - 图像生成
- ElevenLabs - 语音合成

## 最佳实践

### 1. 内容迭代
- 先用AI生成基础框架
- 再根据需求手动调整细节
- 多次生成获取灵感

### 2. 项目组织
```
YourProject/
├── Assets/
│   ├── Scripts/AI_Generated/  # 生成的脚本
│   ├── Resources/Scenes/      # 场景配置
│   └── Resources/Data/         # JSON数据
└── Documentation/              # 设计文档
```

### 3. 版本控制
- 为每个生成的内容添加版本号
- 记录修改历史
- 保留原始生成结果
